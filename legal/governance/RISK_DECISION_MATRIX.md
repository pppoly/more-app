# RISK_DECISION_MATRIX v1.0 (2025-11-30)
1. リスク階層
1.1 R1 低リスク：一般イベント。
1.2 R2 中リスク：未成年含む娯楽、軽い接触機会。
1.3 R3 高リスク：大人数、酒類提供、屋外/スポーツ、深夜。
1.4 R4 禁止・要審査：暴力団、宗教勧誘、性風俗、賭博、医療行為、18禁コンテンツ、虚偽/詐欺、危険行為。

2. 審査/対応
2.1 R1: 自動/簡易審査後公開。
2.2 R2: モデレーション + ランダム人工審査。
2.3 R3: KYC/KYB 強化、手動審査、支払遅延/リザーブ可。
2.4 R4: 原則禁止、例外は法務・リスク承認。

3. 支払・凍結
3.1 R3/R4 は Stripe Connect のリスクポリシーに従い、遅延精算・リザーブ・アカウント凍結を実施。
3.2 Chargeback/Dispute 発生時は R3/R4 を優先審査。

4. ログ/保存
4.1 審査結果・判断理由を 2 年保存（電気通信事業法ログ方針に準拠）。
