# CONTENT_MODERATION_GUIDE v1.0 (2025-11-30)
1. 審査範囲
1.1 タイトル/説明/画像/場所/日時/価格/タグを機械・人工で審査。
1.2 R4（暴力団/性風俗/賭博/宗教勧誘/詐欺/医療無許可/18禁）は禁止。

2. プロセス
2.1 R1-2: 自動＋抽選人工。R3: 人工必須。R4: 禁止。
2.2 不備・リスク検知時は pending/reject とし、理由を通知。

3. 外部通報
3.1 Report → 24-72h でレビュー → 必要に応じ緊急停止/削除。
3.2 証拠を 2 年保存（ログ/スクリーンショット等）。

4. 再開条件
4.1 追加資料/KYC 強化/内容修正が完了し、リスク責任を明示した場合に限り再開可。
