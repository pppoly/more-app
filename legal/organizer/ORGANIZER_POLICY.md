# ORGANIZER_POLICY v1.0 (2025-11-30)
1. KYC/KYB
1.1 主催者は Stripe 要件に従い身元確認・銀行口座登録を完了する。
1.2 高リスクカテゴリでは追加書類/審査/遅延精算を行う。

2. 責任
2.1 内容・行為の責任は主催者に帰属し、参加者への履行・安全配慮義務を負う。
2.2 プラットフォームは中継者であり、主催者の不履行による損害について責任を限定。

3. 活動管理
3.1 情報は正確に表示し、変更時は速やかに更新。
3.2 虚偽・詐欺・宗教勧誘・性風俗・暴力団・18禁・危険行為を禁止。
3.3 取消/中止時は返金条件に従い参加者へ通知・対応。

4. データ
4.1 参加者データは目的外利用禁止、APPI・電気通信事業法の規律を遵守。
4.2 退会/終了時は必要な範囲を除き削除/返却。

5. 支払
5.1 精算は Stripe Connect 経由。Chargeback/詐欺兆候時は遅延・凍結・相殺可。
