# 株式会社創翔モア 利用規約 v1.0 (2025-11-30)
1. 定義  
1.1 「本サービス」は 株式会社創翔モア（創翔モア）のイベント/コミュニティ向けプラットフォーム。  
1.2 「主催者」「参加者」「ユーザー」等は役割に応じ区別。  

2. 適用範囲  
2.1 本規約はユーザーすべてに適用し、日本法を準拠法とする。  
2.2 日本語版を基準とし、EN/ZH は同等内容。  

3. アカウント・KYC/KYB  
3.1 登録情報は真実かつ最新。  
3.2 主催者は Stripe 要件に従い KYC/KYB を実施し、未完了の場合は支払停止・イベント公開制限。  

4. サービス提供・決済/精算  
4.1 本サービスは、イベント/コミュニティの申込受付、決済の受領、返金処理、精算管理等を提供するプラットフォームである。  
4.2 決済は当社が Stripe を通じて受領し、所定の精算サイクルおよび審査・リスク対応に基づき主催者へ精算する。イベント/コミュニティの履行は主催者の責任であり、主催者と参加者間の契約履行は当事者が負う。  

5. 禁止行為・コンテンツポリシー  
5.1 虚偽イベント、詐欺、宗教勧誘、性風俗、暴力団/反社、賭博、医療行為無許可、18禁コンテンツ、危険行為を禁止。  
5.2 APPI・電気通信事業法（外部送信規律/ログ保存）に従い、利用状況ログを保存・開示する場合がある。  

6. 料金・手数料・特商法  
6.1 料金・プラットフォーム手数料は表示どおり。  
6.2 事業者情報は特商法に基づき表示。  
6.3 返金条件は「PAYMENT_POLICY」「REFUND_POLICY」に従う。  

7. 決済・Chargeback・税務  
7.1 返金/キャンセルは Stripe ポリシーに従う。  
7.2 Chargeback 発生時、調査完了まで精算を遅延・凍結可。  
7.3 税務は各当事者が適用法に基づき申告・納税。  

8. 免責・責任限定  
8.1 活動中の事故・傷害は主催者/参加者間で解決し、プラットフォームは中継者として責任を限定。  
8.2 不可抗力・通信障害・第三者行為について責任を負わない。  
8.3 間接損害・逸失利益について責任を負わない。  

9. 知的財産  
9.1 提供コンテンツの権利はユーザーまたは正当権利者に帰属。  
9.2 ユーザーは当社に非独占的利用許諾を付与（サービス運営・広告目的の範囲）。  

10. データ・プライバシー  
10.1 APPI に基づき個人情報を取得・利用・保管し、外部送信規律を遵守。  
10.2 第三者提供・委託・クッキー/ログ利用はプライバシーポリシーに従う。  

11. 契約期間・終了  
11.1 ユーザーはいつでも退会可能。  
11.2 規約違反・リスク高のイベントは停止/削除/凍結。  

12. 準拠法・裁判管轄  
12.1 日本法を準拠法とし、東京地方裁判所を専属的合意管轄とする。  

13. 変更  
13.1 変更時は VERSIONING/CHANGELOG に従い通知し、継続利用で同意とみなす。
